# Quiz_APP

## 🚀 Project Overview
Quiz_APP displays multiple-choice questions one at a time, lets users navigate through them, tracks their score, and shows the final result at the end.
## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
